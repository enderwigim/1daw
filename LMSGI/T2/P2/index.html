Los contenidos de este ejercicios tratan de ilustrar el funcionamiento de las etiquetas de formato de texto con HTML. para ello, se utiliza como base el documento HTML del ejercicio anterior.

Encabezados

Los encabezados tiene como función estructurar el contenido del documento en partes para identificar los apartados del texto. Normalmente, se utiliza un encabezado de nivel 1, que corresponde con el de mayor tamaño, para identificar los apartados principales. Se pueden ir utilizando otros encabezados de menos tamaño para identificar los apartados principales. Se pueden ir utilizando otros encabezados de menor tamaño para indicar los subapartados. Existen 6 tipos de encabezados.
